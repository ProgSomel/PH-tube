Discuss the scope of var , let , and const

var:
Function Scope: Variables declared with var are function-scoped. This means they are accessible within the function in which they are declared or globally if declared outside of any function.Variables declared with the var keyword can NOT have block scope. Variables declared inside a { } block can be accessed from outside the block.{ var x = 2; } // x CAN be used outside the scope We can Redeclare a varibale with var. Redeclaring a javascript variable with var is allowed anywhere in a program

let:
Block Scope: let keyword provide block scope in javascript. Variable declared inside a {} block can not be accessed from outside the block scope. With let redeclaring a variable in the same block is not allowed. To use let variable it must be declared before. We can assign null to a variable

const:
Block Scope: const keyword provide block scope in javascript. Variable declared inside a {} block can not be accessed from outside the block scope. Variable defined with const can not be Redeclared . Variable defined with const keyword can not be Reassigned. const variable must be assigned when declared. After declaration it can't be assigned

Tell us the use cases of null and undefined

null:
null is often used to explicitly indicate that a variable or object property intentionally has no value or is empty. Functions may return null to indicate that they didn't produce a meaningful result or encountered an error.

What do you mean by REST API ?

API - Application Programming Interface. API is like a messenger to transfer data between clinets. API is made to not access the actual system but to access the some part of the system.A REST API (Representational State Transfer API) is a specific type of API that follows a set of architectural principles and constraints. REST API has some methods: GET, POST, PUT/PATCH, DELETE. We can do CRUD operation using REST API.
